KEY FEATURES:
-------------
✓ Word selection with keyboard shortcut (Ctrl+Shift+S)
✓ Automatic translation (Google Translate API)
✓ Context sentence extraction
✓ Sidebar vocabulary management
✓ Search and filter functionality
✓ Import/export data (JSON format)
✓ PDF support
✓ Right-click context menu
✓ Visual feedback notifications
✓ Speech synthesis for pronunciation
✓ Notes and review status tracking

INSTALLATION:
-------------
1. Clone repository
2. Open chrome://extensions/
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select project folder

USAGE:
------
1. Select text on any webpage
2. Press Ctrl+Shift+S (or Cmd+Shift+S on Mac)
3. Word is saved with translation and example sentence
4. View in sidebar by clicking extension icon

DATA STORAGE:
-------------
All data stored locally using Chrome Storage API
No cloud storage or external servers
Privacy-focused design

TRANSLATION:
------------
Default: Google Translate (free, no API key required)
Supports: Youdao, Baidu (requires API keys)
Auto-detects source language
